Coding Camp

Camp Description

Programming is something that is often associated with working indoors. Whether it be from home, a cubicle, or an office. This is an opportunity to learn how to write code from a completely different environment, the forest! We offer a 3 day stay at our camp that includes a basic coding course alongside other fun outdoor activities!

Register Here!

Why learn to code with us?

  • We teach the basic of many different languages
  • Learn introductory Python, Java, HTML/CSS, and more!
  • Partake in activities such as swimming and rock climbing during free time!
  • Code in an environment completely different from the usual!
